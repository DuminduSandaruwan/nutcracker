Nutcracker: RGB Effects Builder

Project proposal:

Develop a web based tool that will allow the easy creation of animations for RGB devices. There will be a number of target models: mega-Trees, matrixes, rays and other shapes. A library of animations (spirals, text, pictures, meteors and many more) can be selected to project onto the target models. The final animation will be exported as an XML file for input for the major sequencing software: Light Show Pro, Vixen or Light-O-Rama.

Method:

The design is platform independent . The plan is for the user to use our models or allow the user to upload their own math models and target models. It is planned to keep this as an open source tool.

The Nutcracker tool has four parts.
Phase# Phase Name Phase Description
1 Target Modeler.
This tool allows the user to model their RGB device (megatree.ray, matrix) and save it to a mysql database. It is very important that your particular tree be defined. This is the only way that an effect can be mapped to your particular display.
2 Effect Generator Here you select an effect from the Library and modify it. effects will be spirals, snowflakes, meteors, scrolling text .etc. Once an effect has been designed it is available to be used on all target sequencers
3 Create a sequence and get an animated GIF. You now select your target model (you might have more than one in your yard) and pick the effects that you want mapped to your display. You will be asked for how long of a duration you want the sequence to run.
4 Create an XML file for a sequencer The last thing you do is select the target sequencer. Currently we have Vixen vir files and LOR S2 lms and lcb files.

To install the nutcracker software on your local PC , you need to follow these steps


I have tested this with a 32bit wamp for Windows XP and a 64 bit wamp for Windows 7
NOTE!! 90% of users will NOT need to install the Nutcracker to their local machine, just run it from the httpp://meighan.net/nutcracker web page. If you install Nutcracker locally, you will be able to bypass the 30 second limit that the meighan.net web site has. If you choose to go ahead and install the Nutcracker locally (even though it probably is not necessary), follow the next steps below.


  • Download the WAMP server software, Click here to download wamp server
  • Download the gnuplot graphing program, Click here to download Gnuplot 4.6
    Note! Be sure you click the last option in this screen, "Add Application dreictory to your Path environment variable"
  • After you have installed WAMP , go get the install script for Nutcracker. Rick click this link Download install.txt After you downloaded it, move it to C:\wamp\www and rename it as install.php
  • Run install.php you just downloaded. enter localhost/install.php into a web window. Installation should take 45 seconds. This process will create a database in your local mySql that is called nutcracker. It will also create a user nc_user/nutcracker123 in your local mySql database. This current version is installing nto the directory nutcracker. This process will also copy down all files needed for Nutcracker to run.
  • launch nutcracker: enter localhost/nutcracker into a web browser. Log in with the same user/password as the meighan.net/nutcracker website.
  • Note! you MUST resave your target models. Do this by clicking on a target and clicking on the submit button. This way a target.dat file is created in the local file system.
  • Now , go to the effects screen and try one of the effects. Try butterfly or spiral. You should see an animated gif.
  • Congratulations, you now have a local copy of Nutcracker. The 30 second limit has been removed.
  • If you want to update your website with the latest code, enter "localhost/nutcracker/update.php"
